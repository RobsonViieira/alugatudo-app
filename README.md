# AlugaTudo 📱

Aplicativo mobile para aluguel de itens entre pessoas.

## 🚀 Objetivo
Conectar pessoas que têm objetos parados com pessoas que precisam usá-los por pouco tempo.

## 📦 Funcionalidades (MVP)
- Cadastro e login
- Cadastro de itens
- Listagem
- Solicitação de aluguel
- Histórico

## 🛠️ Tecnologias
- React Native
- Firebase
- GitHub
